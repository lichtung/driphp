在PHP后端领域玩了4年，稍稍有点疲倦。
搞了自己的个人网站，开始沉淀和新的征程。

WEB技术栈

你现在看到的这个站点基于下面的这些技术

PHP7.x

PHP是WEB领域最流行的开发语言，得益于云计算和硬件水平的发展，以及版本7带来的新特性和性能提升

Bootstrap

最受欢迎的前端框架，本站就是基于Bootstrap4构建前台界面

Linux

主服务器环境基于CentOS7系搭建，从服务器使用相对节约资源的Debian系列

Database

数据的存储使用了MySQL，MongoDB数据库，此外Redis也被用作高速缓存投入了使用

Search Engine

基于Lucene/Solr构建信息检索引擎

H5

应用H5到移动APP，桌面应用中，节约开发成本
